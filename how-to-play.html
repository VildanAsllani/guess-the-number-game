Instructions how to play the game

Guess the number game is a small simple but quite fun game. This game is suitable for spending some free time and does not create any addiction like most games of modern times and with high risk for both mental and physical health.

The main goal of the game is to discover the number that is randomly generate from 1 to 20

    GAME RULES

  1. You only have 5 chances to get the correct number.
  2. You can not select the same number more than once.
  3. When you select the wrong number then the color of the number changes to indicate that the number has been selected.
  4. You have a count of how many times you have tried to find the correct number with all the numbers you have tried.
  5. You will have helping text every time you make a guess.
1) You only have 5 chances to get the correct number

It is simple if you try 5 times and with the help (hints) the message we give you, you are not able to find the number then you will be welcomed with a 'nice' message as a loser.

2) You can not select the same number more than once

To give you a better chance of winning we will not allow you to pick the same number twice.

3) When you select the wrong number then the color of the number changes to indicate that the number has been selected

Visually you would be able to tell what buttons you click because they will be marked with red.

4) You have a count of how many times you have tried to find the correct number with all the numbers you have tried

As we said at the beginning you have in total 5 tries but if you are focused you may lose track on number of tries so we will display that number and the tries right in front of you.

5) You will have helping text every time you make a guess

Each time you make a guess then a message will appear 'higher' or 'lower' helping you to make the next guess giving you a better chance of winning.
